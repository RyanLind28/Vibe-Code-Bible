# Security Vibe Coding Knowledge Base

> Application security, authentication patterns, and data protection structured for AI-assisted development. Build secure by default.

## Structure

```
Security/
├── OWASP-Top-10/            → XSS, injection, CSRF, SSRF, broken access control
├── Auth-Patterns/           → OAuth 2.0, OIDC, JWT best practices, session security
├── Data-Protection/         → GDPR, encryption, PII handling, data minimization
├── API-Security/            → Rate limiting, input validation, CORS, API keys
└── Supply-Chain/            → Dependency auditing, lockfiles, SBOMs, provenance
```
